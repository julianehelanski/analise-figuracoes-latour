# Figuras consolidadas

Pasta que reúne todas as figuras da análise em um único local, com nomes desambiguados por etapa e obra. Criada em 2026-05-23. Cada arquivo é uma cópia do PNG (e do SVG, quando existe) do diretório canônico onde a figura foi gerada.

Total: 35 figuras (PNG); 32 com SVG correspondente (as três figuras do Reinert/AFC saem do R via `png()` device, sem SVG).

Convenções de nome: `<etapa>_<obra-ou-bloco>_<tipo>.png`. As siglas de obra seguem `lab_life` (1986), `sia` (1987), `pandora` (1999), `clarifications` (1996), `recalling_parcial` (corpus parcial Etapa 2), `recalling_integral` (corpus integral Etapa 2-bis), `aime` (2013).

Inventário descritivo completo das figuras: `outputs/inventario_figuras.md`. Blocos LaTeX prontos para incorporar à tese: `outputs/latex/inventario_figuras.tex`.

## Como manter sincronizada

Esta pasta é uma cópia espelhada. Se uma figura canônica for regerada, é preciso re-copiar manualmente, ou rodar o snippet abaixo (parcialmente automatizável):

```bash
# Exemplo: re-sincronizar figuras do passo 4
for f in comparacao_frequencias_tres_obras densidade_militar_sia_pandora \
         frequencia_grupos_pandora_refinada frequencia_grupos_sia_refinada \
         frequencia_grupos_tres_obras_painel rede_cocorrencia_sia; do
  cp "outputs/etapa1/passo4/figuras/${f}.png" "outputs/figuras/etapa1_passo4_${f}.png"
  cp "outputs/etapa1/passo4/figuras/${f}.svg" "outputs/figuras/etapa1_passo4_${f}.svg"
done
```

A pasta `outputs/consolidado/figuras/` é uma versão anterior do mesmo conceito, com nomes menos sistemáticos e cobertura parcial (sem Reinert/AFC e sem AIME). A pasta atual `outputs/figuras/` substitui aquela como referência única.
